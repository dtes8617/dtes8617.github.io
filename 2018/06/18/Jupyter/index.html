Jupyter Notebook

Posted on 2018-06-18 | In 數據學堂 | | Visitors:

Jupyter相當於RMarkdown但又更勝一籌，他可以在一個頁面中書寫文字、即使呈現圖表，這讓程式碼更容易被理解及閱讀，也因此在分析上Jupyter已經有相當多的擁護者。

安裝上最簡單的方式是使用Anaconda，只要輸入conda install jupyter notebook即可。

開啟Jupyter輸入jupyter notebook

管理環境

為了讓Jupyter能管理不同環境，輸入conda install nb_conda

標籤頁除了多Conda來協助管理環境外，在New裡面也有不同的環境選項。要注意的是如果裡面沒有你要找的環境，則需將nb_conda裝在環境中，才會顯示於New的清單裡。

轉換格式

Notebooks是用JSON的格式，為了可以分享給其他人閱讀或使用，可用指令轉換成其他格式(HTML, Markdown, slideshows, etc)。

製作簡報

Jupyter可以直接將頁面製作成簡報，透過以下路徑點選就可以開啟簡報模式。

開啟後可以在各cell點選簡報的形式，分別有幾種形式可以選：

  • Slide：主頁面，可以左右滑動
  • Sub-Slide：副頁面，可以上下滑動
  • Fragment：這個有點類似動畫，會在上一張投影片中呈現
  • Skip：隱藏此投影片
  • Notes：同樣不會在簡報中播放，會作為註記
